# Sales Data Analysis (Excel)

## Overview

This project analyzes sales data using Microsoft Excel to identify trends and insights.

## Analysis Performed

* Sales by Category
* Sales by Region
* Sales trend over time

## Key Insights

* Identified top-performing product categories
* Analyzed regional sales distribution
* Observed sales growth trends

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
